## [Кнопки и ссылки](/src/block/site/default/button) 
### Кнопки
##### Префикс `btn__`
##### Классы:
* `btn__item`
 
##### Модификаторы `is--`:
* `is--xl` (большой размер кнопки)
* `is--sm` (средний размер кнопки)
* `is--xs` (малый размер кнопки)
* `is--invers` (инверсный цвет кнопки)
* `is--white` (светлый цвет кнопки кнопка)

### Ссылки
##### Префикс `link__`
##### Классы `link__item`
